# Dating Master - 恋爱话术大师

一款专为年轻女性设计的"高情商聊天助手" WebApp。

## 功能特点

- 🤖 AI 智能生成高情商回复
- 🎭 10+ 人设可选：黛玉妹妹、高段位海王、狐媚子、甜妹等
- 🖼️ 支持上传聊天截图自动分析
- 📋 一键读取剪贴板并生成回复
- ⭐ 收藏喜欢的话术到语料库
- 🏷️ 场景标签：开场白、搭讪、调侃、撒娇、午夜微醺等
- 💘 主动撩大哥功能
- 🌽 每日土味撩人情话

## 在线体验

👉 [https://zexicrypto.github.io/dating-master](https://zexicrypto.github.io/dating-master)

## 技术栈

- Vue 3 + Vite
- 纯前端实现，无需后端
- 支持 PWA，可添加到手机桌面

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
